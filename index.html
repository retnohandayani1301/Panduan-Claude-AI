Mulai Sekarang

4 langkah mudah
mulai di claude.ai

1

Buka claude.ai di browser

Kunjungi claude.ai dari browser mana saja — Chrome, Safari, Firefox, atau aplikasi mobile.

2

Klik "Sign Up" dan daftar

Daftar menggunakan akun Google atau email. Prosesnya hanya butuh beberapa menit.

3

Langsung ketik pertanyaan

Setelah masuk, langsung ketik pertanyaan atau permintaan apa saja di kolom chat.

4

Tekan Enter — Claude merespons!

Tidak perlu instalasi khusus. Akun gratis sudah sangat cukup untuk belajar dan mencoba semua fitur.

⚡
Keterbatasan
  • Claude bisa membuat kesalahan — verifikasi info penting dari sumber resmi
  • Pengetahuan memiliki knowledge cutoff — info terkini mungkin tidak tersedia
  • Tidak bisa akses internet real-time kecuali fitur web search diaktifkan
  • Tidak menyimpan percakapan antar sesi secara otomatis
🔒
Privasi & Keamanan
  • Jangan masukkan nomor KTP, password, atau data keuangan
  • Percakapan dapat digunakan untuk melatih model AI
  • Gunakan mode Incognito untuk percakapan lebih privat
  • Baca kebijakan privasi Anthropic untuk detail lengkap
⚖️
Penggunaan Bertanggung Jawab
  • Tetap kritis terhadap setiap informasi yang diberikan AI
  • Jangan mengandalkan Claude untuk keputusan medis atau hukum yang kritis
  • Gunakan sebagai alat bantu, bukan satu-satunya sumber kebenaran
  • Gunakan sesuai kebutuhan nyata secara bijak
